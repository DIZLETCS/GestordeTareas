# 🚀 Gestor de Tareas – Backoffice

Aplicación de escritorio diseñada para optimizar la gestión diaria de tareas, notas y respuestas automatizadas en entornos de trabajo operativos.

Pensada para equipos de backoffice, operaciones y gestión administrativa que necesitan orden, velocidad y control en un solo lugar.

---

## 🎯 ¿Qué es Gestor de Tareas?

Gestor de Tareas es una herramienta integral que permite centralizar:

- ✔ Gestión de tareas diarias
- ✔ Organización por categorías
- ✔ Notas rápidas
- ✔ Respuestas automatizadas reutilizables
- ✔ Seguimiento de actividad

Todo en una interfaz simple, rápida y enfocada en productividad.

---

## ⚙️ Funcionalidades principales

### 📋 Gestión de tareas
- Crear, editar y eliminar tareas
- Clasificación por categorías
- Estados dinámicos (pendiente, completado, etc.)
- Vista diaria organizada

### 🗂️ Categorías personalizadas
- Creación y edición de categorías
- Organización visual clara
- Adaptable a distintos flujos de trabajo

### 📝 Notas rápidas
- Registro inmediato de información importante
- Acceso rápido desde la interfaz principal

### 💬 Respuestas automatizadas
- Guardado de respuestas frecuentes
- Uso rápido en atención al cliente
- Ideal para WhatsApp, mail o gestión interna

### 📊 Exportación de datos
- Exportación a Excel
- Información lista para análisis o reportes

### 🧠 Tutorial interactivo
- Guía inicial automática al abrir por primera vez
- Explicación paso a paso del sistema
- Posibilidad de omitir

## 💻 Instalación

1. Descargar el archivo `.exe` desde la sección **Releases**
2. Ejecutar el instalador
3. Abrir la aplicación desde el acceso directo

---

## ⚠️ Advertencias de seguridad

Al tratarse de una aplicación distribuida como `.exe`, es posible que Windows muestre una advertencia de seguridad.

Esto es normal en aplicaciones no firmadas digitalmente.

👉 Para continuar:
- Hacer clic en **"Más información"**
- Luego en **"Ejecutar de todas formas"**
